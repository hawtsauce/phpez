Easy LAMP Stack Installation with a GUI

phpez provides a user-friendly graphical interface (GUI) and a powerful command-line script to make installing Apache, PHP, MySQL, and phpMyAdmin on your Linux system effortless.

(for distros with apt, dnf, pacman, yum & zypper as package manager)

After running the script, if you chose to install the phpez binary, type phpez in the terminal to launch the GUI.

Why Use phpez?

Automated Installation

No more manual commands. phpez automates the entire setup of Apache, PHP, MySQL, and phpMyAdmin, saving you time and reducing errors.

Interactive GUI

For those who prefer a visual approach, phpez offers a clean and intuitive graphical interface to guide you through the installation process.

Developer Tools

Optionally install essential developer tools like Composer and the phpez binary for extended functionality and easier project management.

Smart Detection

phpez intelligently detects existing installations and configurations, prompting you with choices to avoid conflicts and ensure a smooth setup.
